# Agentario v0.4.8

## Что нового

### Added

- Индексация: выбор embedding-модели LM Studio, UI «Индексация кода».
- Документация LM Studio: domain type Embedding, headless-сервер ([config/lmstudio-indexing.md](config/lmstudio-indexing.md)).
- Скрипт `scripts/lmstudio-headless-server.cmd` — LM Studio без GUI.
- Скрипт `publish-release.cmd` — публикация на GitHub + GitHub Release с VSIX.
- Документация релизов: [config/RELEASE.md](config/RELEASE.md), шаблон `release/notes/TEMPLATE.md`.

### Changed

- Embedding LM Studio: native `/api/v0/embeddings`, резолв loaded instance id, без повторной загрузки модели.
- Раннее предупреждение, если выбрана llm-модель вместо embedding-слота.
- Global rules / system prompt: краткие сообщения в чате; таймаут tools 120 с для LM Studio/Ollama.

### Fixed

- Критический баг 0.4.0: `StateManager is not defined` — запросы к модели не уходили.
- UI: сброс «Thinking» после ошибки/cancel.
- LM Studio: корректный выбор локальной модели вместо каталога по умолчанию.

## Установка

1. Скачайте **`agentario-0.4.8.vsix`** из Assets этого релиза.
2. VS Code → Extensions → `...` → **Install from VSIX**.

## Требования

- VS Code 1.84+
- LM Studio или Ollama для локальных моделей
- Для семантической индексации — embedding-модель в LM Studio (type `embeddings`), см. [lmstudio-indexing.md](config/lmstudio-indexing.md)
