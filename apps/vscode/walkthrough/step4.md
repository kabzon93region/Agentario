# Unlock Specialized Capabilities with MCP

**The Model Context Protocol (MCP) connects Agentario to a world of powerful tools.**

Go beyond local code. With the Model Context Protocol (MCP), Agentario accesses vital context from external datasources like databases and APIs. Configure local or remote MCP servers to give Agentario specialized, secure tools for complex, real-world development tasks.
