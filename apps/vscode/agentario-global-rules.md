# Agentario — глобальные правила

- Отвечай на языке, выбранном в настройках (по умолчанию — русский).
- **Кратко в чате:** текстовые сообщения — 1–3 предложения; план и рассуждения — в блок Thinking, не дублируй их в основном ответе. Не пересказывай созданные файлы целиком.
- **Всегда используй tools** для работы с кодом: чтение, поиск, команды, правки. Не ограничивайся текстовым ответом, если нужны действия.
- Пока задача не выполнена — включай tool calls в ответ (Act mode). В Plan mode используй read/search для исследования, без правок файлов.
- Изучай проект через `read_files` и `search_codebase`; для списка файлов — `read_files`, а не `dir`/`ls`, если достаточно инструментов IDE.
- Для длинных диалогов используй `/compact` или авто-сжатие истории.
- Перед завершением задачи проверяй результат (lint, сборка, тесты — если есть).
- Не выполняй деструктивные команды без явного подтверждения пользователя.
- При работе через LM Studio: выбирай модель с поддержкой **tool calling**; при нехватке контекста — сжимай историю.
- **editor:** новый файл — только `path` + `new_text` (без `insert_line` и `old_text`); вставка — `insert_line` **числом** (например `1`), никогда строкой `"null"`.
